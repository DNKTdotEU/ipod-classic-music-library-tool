# Third-Party Attribution Guidance

## Dependency attribution
- Third-party package licenses are managed through `package.json` dependencies.
- Before each release, review license compatibility for newly added dependencies.
- Keep NOTICE and license references accurate when adding bundled third-party assets.

## Media and metadata sources
- External metadata and artwork APIs may have attribution requirements.
- Preserve required attribution where mandated by upstream providers.

## Release check
- Include attribution review in release checklist and PR review for dependency changes.
