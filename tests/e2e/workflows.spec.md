# E2E Scenario Specification

This project includes scenario definitions for desktop automation:

1. Scan -> duplicate list -> compare -> keep -> verify dashboard counters.
2. Quarantine move -> restore -> verify history event.
3. Batch metadata normalization preview -> apply -> verify resulting fields.
4. Artwork search by album/artist -> select candidate -> embed confirmation.

These specs are intentionally deterministic and paired with `fixtures/golden-dataset.json`.
